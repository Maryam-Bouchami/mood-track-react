Mood Track:
Mood Track is a simple React application that lets users record their daily mood by selecting a date and choosing a mood button.
The app displays mood history through two charts â€” a line chart (mood score over time) and a donut chart (mood distribution). 
Users can also clear their entire history.
-----------------------------------------------------------------------------------------------------------------------------------

Features :

Select a date and save your current mood

View mood history on a line chart and donut chart

Clear all saved moods with Clear History

Highlighted active mood button

-----------------------------------------------------------------------------------------------------------------------------------
Installation:
git clone https://github.com/Maryam-Bouchami/mood-track-react.git

cd mood-track-react

npm install

npm run dev
